# Apresentação do projeto de portfólio

Este é o segundo tipo de exercício que você terá nesta trilha. Trata-se de um projeto de portfólio, e serve para ilustrar o que você aprenderá durante a trilha.

Gostaríamos que você mudasse o nome Roseli Silva para seu próprio nome (ou qualquer outro nome que desejar). Este nome está presente no elemento h1.

Obs: embora você possa encontrar o nome Roseli no restante da página, não é necessário alterá-lo em todos os lugares, basta alterar o texto do h1.

## Testes 
- Altere o texto Roseli Silva no h1 para qualquer outro nome.
