# Strings com múltiplas linhas

- Não é possivel ter uma quebra de linha usando aspas simples ou duplas.
- É usado o **``** se for utilizar uma quebra de linha no texto.

```
let mensagem = `Bem-vindo ao
nosso site!`
```

## Exercício - Mensagem longa de múltiplas linhas

Crie uma variável chamada mensagem e armazene dentro dela uma string que abrange várias linhas. Você pode escrever o que quiser, desde que essa string ocupe várias linhas. Sinta-se à vontade para acessar console.log(mensagem) no final, para exibir essa string no console.

Implemente a mensagem com várias linhas. E, se quiser, faça um log da variável mensagem.

### Testes
- Lembre-se de utilizar uma template string para que as várias linhas sejam aceitas pela variável.
