# Introdução ao controle de versão

- Versionamento é a maneira como cuidamos das versões de um código.
- Git é um sistema de controle de versão.
- Github é uma ferramenta que provê acesso ao Git.

## Começando com o GitHub

- GitHub desktop é um software cliente para manipular o GitHub.
- Existem outras formas de manipular o github, como o terminal.
- Repositório é o local onde o código fica armazenado.
- Clonar é o comando usado para baixar todos os arquivos para o computador.

## Criando um repositório

- Existe mais de uma forma de criar um repositório.
- O nome do repositório deve ser escrito em letras minúsculas e com as palavras separadas por hífen.

## Clonando um repositório e subindo o primeiro arquivo

- Clonar um repositório signifca baixar para o comptador todos os arquivos do repositório que estão online.
