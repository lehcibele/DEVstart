# Por que precisamos de objetos?

- Um objeto é um tipo de dados que permite agrupar várias variáveis em uma única variável que contém chaves e valores.
- Também podemos definir um objeto como uma coleção de pares chave/valor.
- Objetos são diferentes de matrizes, porque as matrizes contêm chaves e valores.

```
let pessoa = {
    primeiroNome: "Silvio",
    sobrenome: "Duarte",
    idade: 21,
    email: "silvioduarte@gmail.com",
    estaVerificado: true
};
```
