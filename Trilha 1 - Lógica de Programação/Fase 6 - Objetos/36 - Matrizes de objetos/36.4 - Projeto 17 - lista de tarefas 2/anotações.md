# Projeto 17 - Lista de tarefas II

Complete as funções da lista de tarefas conforme explicado a seguir.

criarTarefa(): Exiba o objeto dados no console e complete a função de maneira que ela retorne um objeto com a seguinte estrutura:
{
    titulo: "Minha tarefa", // título da tarefa
    estaCompleta: true, // booleano indicando se a tarefa está completa ou não
    categoria: "trabalho", // categoria da tarefa
    autor: { // objeto aninhado contendo os dados do autor
        nome: "Alice",
        sobrenome: "Fernandes"
    }
}
adicionarTarefa(): Esta função recebe a matriz de tarefas e a nova tarefa a ser adicionada. Então, ela retorna a matriz atualizada (após a tarefa ter sido adicionada a ela).
conteTarefa(): Deve retornar o número de tarefas na matriz.
primeiraTarefaTitulo(): Deve retornar o título da primeira tarefa.
ultimaTarefaTitulo(): Deve retornar o título da última tarefa.
exportacaoDeTitulos(): Deve retornar uma matriz contendo os títulos das tarefas que estão na matriz tarefas.
Complete as funções conforme especificado.

## Testes

- Implemente a função criarTarefa().
- Implemente a função adicionarTarefa().
- Implemente a função conteTarefa().
- Implemente a função primeiraTarefaTitulo().
- Implemente a função ultimaTarefaTitulo().
- Implemente a função exportacaoDeTitulos().
