# Projeto 15 - Lista de compras

Complete as duas funções que compõem o aplicativo de lista de compras:

adicionarItem() deve adicionar o item que recebe como parâmetro à matriz de itens que também é recebida como parâmetro. Depois, deve retornar essa matriz.
exportarMinusculasCSV() deve retornar uma string contendo todos os itens da matriz recebida como parâmetro. Nessa string, os itens devem estar em letras minúsculas e separados por uma vírgula e um caractere de espaço.
Siga as instruções acima.

### Testes
- Implemente a função adicionarItem().
- Implemente a função exportarMinusculasCSV().
