# Matriz como parâmetro de função I

## Função que recebe a matriz como parâmetro

```
function obterPrimeiroItem(itens) {
    return itens[0]
}

obterPrimeiroItem([6, 2, 5]) // 6
obterPrimeiroItem([4, -3, 8]) // 4
```
